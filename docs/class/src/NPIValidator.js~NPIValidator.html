    NPIValidator

    You can directly use instance of this class. nPIValidator

    Extends:

    Transform → NPIValidator

    Stream transform to validate NPIs. Validates NPI strings piped through a stream.

    Example:

    
import split from 'split'
import {NPIValidator} from 'npi'

const validator = new NPIValidator()

process.stdin
.pipe(split())
.pipe(validator)
.pipe(process.stdout)
